### 2.1.6 Shadowing: –ø–µ—Ä–µ–≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è –∑–º—ñ–Ω–Ω–æ—ó

**Shadowing** ‚Äî —Ü–µ –º–µ—Ö–∞–Ω—ñ–∑–º, —è–∫–∏–π –¥–æ–∑–≤–æ–ª—è—î –æ–≥–æ–ª–æ—Å–∏—Ç–∏ –Ω–æ–≤—É –∑–º—ñ–Ω–Ω—É –∑ —Ç–∏–º —Å–∞–º–∏–º —ñ–º'—è–º, —â–æ –π —ñ—Å–Ω—É—é—á–∞. –ù–æ–≤–∞ –∑–º—ñ–Ω–Ω–∞ "–∑–∞—Ç—ñ–Ω—é—î" (shadows) —Å—Ç–∞—Ä—É.

```rust
    println!("–ü–µ—Ä—à–∞ x: {}", x);
    
    let x = x + 1;  // –ù–æ–≤–∞ –∑–º—ñ–Ω–Ω–∞ x, —â–æ –ø—Ä–∏—Ö–æ–≤—É—î —Å—Ç–∞—Ä—É
    println!("–î—Ä—É–≥–∞ x: {}", x);
    
    let x = x * 2;  // –©–µ –æ–¥–Ω–∞ –Ω–æ–≤–∞ –∑–º—ñ–Ω–Ω–∞ x
    println!("–¢—Ä–µ—Ç—è x: {}", x);

```

**–©–æ —Ç—É—Ç –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è –ø–æ–∫—Ä–æ–∫–æ–≤–æ:**

### 2.1.7 –ö–ª—é—á–æ–≤–∞ —Ä—ñ–∑–Ω–∏—Ü—è: mut vs shadowing

**`mut` ‚Äî –∑–º—ñ–Ω—é—î –∑–Ω–∞—á–µ–Ω–Ω—è —Ç—ñ—î—ó —Å–∞–º–æ—ó –∑–º—ñ–Ω–Ω–æ—ó:**
