### Shadowing (–∑–∞—Ç—ñ–Ω–µ–Ω–Ω—è)

–£ Rust –º–æ–∂–Ω–∞ –æ–≥–æ–ª–æ—Å–∏—Ç–∏ –Ω–æ–≤—É –∑–º—ñ–Ω–Ω—É –∑ —Ç–∏–º —Å–∞–º–∏–º —ñ–º'—è–º ‚Äî —Ü–µ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è shadowing:

```rust

    println!("x = {}", x);  // 5
    
    let x = x + 1;          // –ù–æ–≤–∞ –∑–º—ñ–Ω–Ω–∞ x "–∑–∞—Ç—ñ–Ω—è—î" –ø–æ–ø–µ—Ä–µ–¥–Ω—é

    println!("x = {}", x);  // 12
```


### –ó–∞–≤–¥–∞–Ω–Ω—è 2: –û–±—á–∏—Å–ª–µ–Ω–Ω—è –≤—ñ–¥—Å—Ç–∞–Ω—ñ
